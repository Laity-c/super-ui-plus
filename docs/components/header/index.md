## API

### Props

| 参数 | 说明 | 类型 | 可选值 | 默认值 |
| --- | --- | --- | --- | --- |
| divider | 是否显示分割线 | `boolean` | - | `false` |
| size | 尺寸 | `string` | `small` / `medium` | `small` |
| border | 底部边框线 | `boolean` | - | `false` |

### Slots

| 插槽名 | 说明 |
| --- | --- |
| default | 标题内容 |
| right | 右侧扩展插槽 |
